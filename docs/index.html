10-Minute Auto Sync

Runs every 10 minutes via GitHub Actions. Automatically fetches token data from DEX APIs, creates snapshots, and archives old data.

  • 144 snapshots per coin daily
  • 8-minute timeout (safe)
  • Auto-retry on failure

2-Year Data Retention

Keep 2 years of rolling historical data with automatic archival and cleanup. No manual action required.

  • ~50 GB for 1000 tokens
  • Automatic archival daily
  • Search & sort instantly
